# Figures

`chexpert_small_frontal1000_torchxrayvision/` contains metric-only figures
derived from the committed aggregate metric table. It contains no X-ray images.

This directory is reserved for figures generated from real MedShiftLab-CXR experiment outputs.

Examples:

- AUROC/AUPRC summary plots
- Brier/ECE calibration summary plots
- reliability diagrams
- dataset summary plots

Do not add synthetic or illustrative figures as scientific results.
