# CheXpert uncertainty comparison figures

These figures visualize the real metadata summaries across uncertainty strategies; they do not show model inference results.

- `mean_target_by_uncertainty_strategy.png`: compares each label's mean target under the four strategies.
- `positive_prevalence_by_uncertainty_strategy.png`: compares positive prevalence by label and strategy.
- `soft_counts_by_label.png`: shows the number of soft targets retained for each label.

See the [results README](../../results/real_runs/chexpert_train_chexbert_uncertainty_comparison/README.md) for dataset details, interpretation, and non-claims.
